# Dictate - Video-conferencing for small meetings and groups

## Commands

### Build

```console
$ python manage.py build
```

### Run

```console
$ python manage.py runserver
```

### Endpoints

- /
  - main app
- /api/
  - api endpoints
- /api-auth/
  - logging into api for accessing /api/ through browser
    - /api-auth/login
    - /api-auth/logout
